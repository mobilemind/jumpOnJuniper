jump On Juniper v2.9g

These steps make a bookmark that will fill-in the VisitorNet Username & password and submit the login. See Notes & Troubleshooting for more information.

Bookmark use: Quick version- Follow instruction below. Then connect WiFi to VisitorNet, then tap the bookmark repeatedly until you are logged in. You'll need to wait between taps to allow the initial sign-in and/or “accept/decline” page to load.

Step 1: Enter the text

Enter the Username to paste and password. Then tap Make jOJ Bookmark to reload this page for Step 2.

  

   

Bookmark as link: jumpOnJuniper

Step 2: Add a Bookmark

Tap the action button on the toolbar. Then select Add Bookmark

Take note of the name (or edit it), and then tap Save.

Step 3: Edit the Bookmark

First, tap on the text box with Code for your bookmark. Then Select all of the text and Copy it to the clipboard.

Now edit the bookmark you added in step 2 to turn it into a javascript bookmark. Tap the Bookmarks button in the toolbar.

Navigate to the bookmark you just saved. Do not select the bookmark. Tap Edit to make changes.

In the “Edit” pop-over, tap on the bookmark name.

In the Edit Bookmark pop-over, tap in the field that begins with “http: ”. It is easiest to replace all of it with the contents of the clipboard, so simply tap the circle with an X in it. Refer to image below.

Finally, tap and hold in the now empty Address field and select Paste. The updated book mark appears. The end of it looks like:

 … ;void('v2.9g')

I wouldn't recommend doing this because the bookmark code is so long, but if you were to scroll to the start you'd see the bookmark begins:

 javascript:var%20u= …

Step 4: Save the Bookmark

Back-up a level in the edit pop-over by tapping the button to the left of “Edit Bookmark”.

Save the bookmark by tapping Done.

Using jumpOnJuniper

Connect to a VisitorNet WiFI access point. Tap the jOJ bookmark once to go to VisitorNet sign-in page, then tap it again to “paste” the stored username and password. Finally, after the screen updates, tap the bookmark a third time to automagically scroll down and accept the Terms.

PRO TIP

Use the Settings app — Safari settings to set Always Show Bookmarks Bar as ON and save (or move) the jOJ bookmark to the Bookmarks Bar folder. Then you can more quickly tap the link for your login.

Troubleshooting

  • Ensure your bookmark is the latest version. Look at the last view characters of the bookmark “address” for the version number found between parenthesis. For example, …void('v2.9g') would indicate you bookmark is current against the title of this page, jumpOnJuniper v2.9g.
  • Force a refresh of this page by holding down the shift key and clicking on the browser refresh button. If the title of the page changes to a new number, then your browser had cached an older version of this page. Make a new bookmark or update the old one using the new, reloaded page.
