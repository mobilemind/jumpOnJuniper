jump On Juniper

These steps make a bookmark that will fill-in the VisitorNet Username & password and submit the login. When creating multiple bookmarks, press reset in-between doing each one.

If you're unsure about entering your password; don't use this bookmark.

Step 1: Enter the text

Enter the Username to paste and password. Then tap Make jOJ Bookmark to reload this page for Step 2.



   

Bookmark as link: jumpOnJuniper

Step 2: Add a Bookmark

Tap the add button on the toolbar. Then select Add Bookmark

Take note of the name (or edit it), and then tap Save.

Step 3: Edit the Bookmark

Now edit the bookmark to turn it into a Pastelet. Tap the Bookmarks button in the toolbar.

Navigate to the bookmark you just saved. Do not select the bookmark.

Tap Edit to make changes.

In the “Edit” pop-over, tap on the bookmark name.

In the Edit Bookmark pop-over, tap in the field that begins with “http:”. You need to edit the start of the text, so tap and then drag all the way to the left.

Place the edit point between the “?” and the “j”. Then tap the delete key until the edit point is at the start of the line. When done, the bookmark should look like:

Step 4: Save the Bookmark

Back-up a level in the edit pop-over by tapping the button to the left of “Edit Bookmark”.

Save the bookmark by tapping Done.

Using jumpOnJuniper

Open the VisitorNet sign-in page, then “paste” the stored Username and password by selecting the “jOJ login…” bookmark. Just tap the bookmarks icon.

Then tap on the Pastelet bookmark to paste in the Username, check the “Agree to Terms” box, complete the password field, and login.

PRO TIP

Use the Settings app — Safari settings to set Always Show Bookmarks Bar as ON and save (or move) the VNet+P login bookmark to the Bookmarks Bar folder. Then you can more quickly tap the link for your login.
